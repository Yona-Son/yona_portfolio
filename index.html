웹퍼블리셔 손화연 입니다 : )

안녕하세요 신입 웹퍼블리셔 손화연 입니다.
새로운 것을 두려워하지 않는 자세와 좋아하는 것은 더 잘하고 싶은 욕심이 있습니다.
웹 표준을 준수한 코딩, 시멘틱 웹을 고려한 마크업과 다양한 플러그인을 응용한 생산성 있는
인터렉션 구현, 모바일 환경에 대응하는 반응형 제작을 할 수 있는 실력을 갖췄습니다.
만화를 전공하여 미감과 연출에 대한 이해도가 높고 관심이 많으며 직접 드로잉한 일러스트로
시각적인 연출에도 자신있습니다.

원활한 협업을 위해 무엇보다 소통을 중요하게 생각합니다.
책임감을 가지고 즐겁게 UI 개발을 하고싶습니다!
